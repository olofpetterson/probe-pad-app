Troubleshooting

⏳

Request times out

Check your internet connection and verify the API endpoint is correct. Some APIs may have rate limiting.

🔒

Can't access Secrets Vault

Ensure Face ID / Touch ID is enabled in your device settings. You can also use your device passcode as fallback.

📥

Import failed

Make sure your Postman export is in Collection v2.1 format. Other formats are not currently supported.
